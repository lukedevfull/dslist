# Título do Seu Projeto

## Descrição

Projeto full-stack de uma plataforma de jogos, construída com as seguintes tecnologias:

Backend: Java com Spring Boot, oferecendo uma API REST para gerenciar jogos, usuários e outras funcionalidades.
Banco de dados: PostgreSQL para armazenamento persistente de dados e H2 para desenvolvimento e testes.
Frontend: HTML, CSS e JavaScript para criar uma interface intuitiva.
O projeto inclui funcionalidades como:

Criação, edição e exclusão de jogos dentro da coleção

## Tecnologias Utilizadas

* **Backend:** Java, Spring Boot, PostgreSQL, H2
* **Frontend:** HTML, CSS, JavaScript
* **Outras:** Postman
